## Technology stack

- **UI**: `Vue 3`
- **Data model**: `Vuex`
- **Routing**: `vue-router`
## Project setup
```
npm install
```

Insert API key from https://github.com/settings/tokens/ in the `.env` file

```
API_KEY=YOUR_API_KEY
```

### Compiles and hot-reloads for development
```
npm run start
```

### Development mode
```
npm run build:dev
```

### Production mode
```
npm run build:prod
```
